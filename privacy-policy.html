Privacy Policy

Effective date: May 29, 2026 · Last updated: May 29, 2026

1. Overview

This Privacy Policy explains what data is collected and how it is handled by the content automation tool ("Auto-Tube") operated by Arman ("we," "us," "operator") in the production of the YouTube channel Redacted Chapter ("the Channel"). Auto-Tube is a personal-use tool, not a public service. It does not have user accounts and does not serve any audience other than the operator.

If you are a viewer of Redacted Chapter content on YouTube, TikTok, or any other platform, that viewing relationship is governed by the privacy policies of those platforms, not by this document.

2. Who this policy is for

This policy is provided to satisfy the disclosure requirements of third-party developer platforms (YouTube Data API, TikTok Content Posting API, Anthropic API, ElevenLabs API, and similar) whose terms require operators of integrating applications to publish a privacy policy. It also serves as a transparency statement for anyone who wants to understand how the Channel is produced.

3. What data we collect

3.1 OAuth credentials (operator only)

To publish to YouTube and TikTok on behalf of the Channel owner, Auto-Tube stores OAuth refresh tokens issued by those platforms. These tokens are:

  • Stored locally in a PostgreSQL database on operator-controlled infrastructure.
  • Encrypted at rest using application-level symmetric encryption (Fernet / AES-128-GCM).
  • Used solely to authenticate uploads to the operator's own channel and account.
  • Never transmitted to any third party other than the platform that issued them.

3.2 Cached API responses

Auto-Tube caches responses from the APIs it consumes (research sources, stock-media APIs, text-to-speech provider, LLM provider) to avoid redundant calls. These caches are stored locally and contain only the API's response data — not personal data about end users of the Channel.

3.3 Operational logs

Auto-Tube writes structured application logs for debugging. Logs do not include OAuth tokens, API keys, or personally identifiable information. Logs are retained on operator-controlled infrastructure only.

3.4 What we do NOT collect

  • We do not collect, store, or process any data about viewers of the Channel.
  • We do not use cookies, analytics SDKs, or trackers on any Auto-Tube interface (Auto-Tube has no public web interface).
  • We do not collect information from third parties about the operator beyond what the platform OAuth flows return.
  • We do not knowingly collect information from children under 13.

4. Third-party services used in production

To produce Channel content, Auto-Tube makes API calls to the following third parties. Each operates under its own privacy policy:

  • Anthropic — LLM script generation (claude.com)
  • ElevenLabs — text-to-speech narration (elevenlabs.io)
  • YouTube Data API — video upload + metadata (google.com)
  • TikTok Content Posting API — vertical-short upload (tiktok.com)
  • Pexels, Pixabay, Tenor — licensed stock and reference media
  • Public-record sources — SEC EDGAR, court records, archived news (where lawful)

Auto-Tube sends only the data each API requires to perform its function (e.g., narration text to ElevenLabs, video file + metadata to YouTube). It does not share OAuth tokens, API keys, or operator identity across services.

5. How long we keep data

  • OAuth tokens: kept until the operator revokes them via the issuing platform or deletes the local database row.
  • Cached API responses: kept until manually pruned or the underlying source ages out (typically 7–90 days).
  • Application logs: rotated on operator-controlled infrastructure; not exported.

6. Data sharing and disclosure

We do not sell, rent, or share data with third parties beyond what is necessary to publish Channel content via the platform APIs listed in Section 4. We do not run advertising and do not participate in ad networks.

We will disclose information only if required by a valid legal process (subpoena, court order) and only to the extent required by that process.

7. Your rights

Because Auto-Tube does not collect viewer data, end-user rights requests do not apply to this tool directly. Viewer rights regarding YouTube, TikTok, and other platforms are governed by those platforms' privacy policies.

The operator can at any time revoke OAuth grants in the YouTube ("Apps with access to your account") or TikTok ("Manage app permissions") settings; doing so immediately invalidates the stored refresh token.

8. Security

Auto-Tube runs on operator-controlled infrastructure. OAuth tokens are encrypted at rest. API keys are stored only in environment variables and never logged. No production credentials are checked into source control. The operator is the sole user of the system.

9. International users

Auto-Tube does not process viewer data, so cross-border data transfer rules (GDPR, UK GDPR, CCPA, etc.) do not apply at the tool level. Cross-border behavior of YouTube, TikTok, and other platforms is governed by their respective policies.

10. Children's privacy

Auto-Tube has no user accounts and does not collect data from viewers, including minors. The Redacted Chapter Channel is not directed at children under 13. Content is investigative business journalism intended for general adult audiences.

11. Changes to this policy

We may update this policy. Any change will be reflected in the "Last updated" date at the top of this document. Material changes that affect how the operator's data is handled will be reflected in a new effective date.
