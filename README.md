# auto-tube-legal

Public legal documents for the **Redacted Chapter** YouTube channel and the Auto-Tube content automation tool that produces it.

Hosted via GitHub Pages.

- [Privacy Policy](https://arm-yan.github.io/auto-tube-legal/privacy-policy.html)
- [Terms of Use](https://arm-yan.github.io/auto-tube-legal/terms-of-use.html)

These URLs are referenced by:

- TikTok Content Posting API developer application
- YouTube Data API developer application (when re-verification is required)
- Any future API integration that requires a public privacy policy or terms of service

## Updating

Edit the HTML files in this repo, commit, push. GitHub Pages rebuilds automatically within ~30 seconds.

Bump the "Last updated" date on any document you change. Bump the "Effective date" only for material changes that affect data handling or user obligations.
