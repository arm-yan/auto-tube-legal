Auto-Tube

Content automation tool for the Redacted Chapter YouTube channel.

About Redacted Chapter

Redacted Chapter is an independent YouTube channel publishing long-form investigative documentaries about corporate failure, regulatory action, deception, and business collapse. Editorial standards favor primary sources (SEC filings, court records, archived journalism, earnings transcripts) over speculation. The tone is calm, evidence-based, and restrained — closer to MagnatesMedia or ColdFusion than typical commentary content.

About Auto-Tube

Auto-Tube is the internal automation system that researches, scripts, narrates, sources visuals, renders, and publishes Redacted Chapter episodes and short-form derivatives. It is a single-user tool used only by the channel operator. There are no end-user accounts, no public-facing web interface beyond this page, and no audience access to the system itself.

How Auto-Tube uses platform APIs

  • YouTube Data API — uploads finished long-form documentaries (10–15 min) and vertical short-form clips (30–60 sec) to the operator's own YouTube channel as Unlisted by default. The operator manually flips visibility to Public.
  • TikTok Content Posting API — uploads vertical short-form derivatives (30–60 sec, 1080×1920) to the operator's own TikTok account. Defaults to private (mutual-friend visibility) until the operator manually adjusts settings inside the TikTok app.
  • Anthropic, ElevenLabs, Pexels, Pixabay, Tenor — used during production to generate scripts, narration, and sourced visuals. None of these post anything externally.

Auto-Tube does not access viewer data, comments, or analytics from any platform. It only publishes content owned and authored by the channel operator.
